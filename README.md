# 🐱 Jogo da Memória - Gatinhos

Um jogo da memória divertido e interativo com tema de gatinhos, construído com HTML, CSS e JavaScript puro.

🔗 **Jogue agora:** [https://flap.github.io/jogo-memoria-gatinhos/](https://flap.github.io/jogo-memoria-gatinhos/)

---

## Sobre o Projeto

O Jogo da Memória - Gatinhos é um jogo clássico de encontrar pares, onde o jogador deve virar as cartas e memorizar suas posições para encontrar todos os pares de emojis de gatinhos. O jogo conta com animações suaves, design responsivo, 3 níveis de dificuldade e suporte bilíngue (Português/Inglês).

---

## Funcionalidades

- **Suporte Bilíngue (PT-BR / EN):**
  - 🇧🇷 Português (padrão)
  - 🇺🇸 English
  - Troca de idioma por bandeiras no canto superior direito

- **3 Níveis de Dificuldade:**
  - 😺 **Fácil** — Grid 3x4 com 6 pares (12 cartas)
  - 😼 **Médio** — Grid 4x4 com 8 pares (16 cartas)
  - 🙀 **Difícil** — Grid 4x5 com 10 pares (20 cartas)

- **Interface Interativa:**
  - Animação de virar cartas em 3D
  - Botões de dificuldade com cores distintas (verde, laranja, vermelho)
  - Mensagem de vitória animada com estatísticas

- **Contador de Desempenho:**
  - Contador de movimentos
  - Contador de pares encontrados
  - Cronômetro de tempo

- **Controles do Jogo:**
  - Novo Jogo — embaralha as cartas e reinicia
  - Reiniciar — mantém a mesma disposição e reseta o progresso

- **Design Responsivo:**
  - Funciona em desktop, tablet e celular
  - Grid e tamanho das cartas se adaptam ao dispositivo

---

## Tecnologias

- HTML5
- CSS3 (Flexbox, Grid, Animações, Gradientes)
- JavaScript (ES6+)
- GitHub Pages (deploy)

---

## Como Jogar

1. Escolha o idioma clicando na bandeira 🇧🇷 ou 🇺🇸
2. Escolha o nível de dificuldade (Fácil, Médio ou Difícil)
3. Clique em uma carta para virá-la
4. Clique em outra carta para tentar encontrar o par
5. Se as cartas forem iguais, elas permanecem viradas
6. Se forem diferentes, elas voltam ao estado original
7. Encontre todos os pares para vencer o jogo!

# 🐱 Memory Game - Kittens (English)

A fun and interactive memory game with a kitten theme, built with pure HTML, CSS, and JavaScript.

🔗 **Play now:** [https://flap.github.io/jogo-memoria-gatinhos/](https://flap.github.io/jogo-memoria-gatinhos/)

---

## About the Project

Memory Game - Kittens is a classic pair-matching game where the player flips cards and memorizes their positions to find all matching pairs of cat emojis. The game features smooth animations, responsive design, 3 difficulty levels, and bilingual support (Portuguese/English).

---

## Features

- **Bilingual Support (PT-BR / EN):**
  - 🇧🇷 Portuguese (default)
  - 🇺🇸 English
  - Language switch via flag buttons in the top-right corner

- **3 Difficulty Levels:**
  - 😺 **Easy** — 3x4 grid with 6 pairs (12 cards)
  - 😼 **Medium** — 4x4 grid with 8 pairs (16 cards)
  - 🙀 **Hard** — 4x5 grid with 10 pairs (20 cards)

- **Interactive Interface:**
  - 3D card flip animation
  - Color-coded difficulty buttons (green, orange, red)
  - Animated victory message with statistics

- **Performance Tracking:**
  - Move counter
  - Pairs found counter
  - Time tracker

- **Game Controls:**
  - New Game — shuffles cards and restarts
  - Reset — keeps the same layout and resets progress

- **Responsive Design:**
  - Works on desktop, tablet, and mobile
  - Grid and card sizes adapt to the device

---

## Technologies

- HTML5
- CSS3 (Flexbox, Grid, Animations, Gradients)
- JavaScript (ES6+)
- GitHub Pages (deployment)

---

## How to Play

1. Choose your language by clicking the 🇧🇷 or 🇺🇸 flag
2. Select the difficulty level (Easy, Medium, or Hard)
3. Click on a card to flip it
4. Click on another card to try to find a match
5. If the cards match, they stay flipped
6. If they don't match, they flip back
7. Find all pairs to win the game!
