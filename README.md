# 🐱 Jogo da Memória - Gatinhos

Um jogo da memória divertido e interativo com tema de gatinhos, construído com HTML, CSS e JavaScript puro.

🔗 **Jogue agora:** [https://flap.github.io/jogo-memoria-gatinhos/](https://flap.github.io/jogo-memoria-gatinhos/)

---

## Sobre o Projeto

O Jogo da Memória - Gatinhos é um jogo clássico de encontrar pares, onde o jogador deve virar as cartas e memorizar suas posições para encontrar todos os pares de emojis de gatinhos. O jogo conta com animações suaves, design responsivo e 3 níveis de dificuldade.

---

## Funcionalidades

- **3 Níveis de Dificuldade:**
  - 😺 **Fácil** — Grid 3x4 com 6 pares (12 cartas)
  - 😼 **Médio** — Grid 4x4 com 8 pares (16 cartas)
  - 🙀 **Difícil** — Grid 4x5 com 10 pares (20 cartas)

- **Interface Interativa:**
  - Animação de virar cartas em 3D
  - Botões de dificuldade com cores distintas (verde, laranja, vermelho)
  - Mensagem de vitória animada com estatísticas

- **Contador de Desempenho:**
  - Contador de movimentos
  - Contador de pares encontrados
  - Cronômetro de tempo

- **Controles do Jogo:**
  - Novo Jogo — embaralha as cartas e reinicia
  - Reiniciar — mantém a mesma disposição e reseta o progresso

- **Design Responsivo:**
  - Funciona em desktop, tablet e celular
  - Grid e tamanho das cartas se adaptam ao dispositivo

---

## Tecnologias

- HTML5
- CSS3 (Flexbox, Grid, Animações, Gradientes)
- JavaScript (ES6+)
- GitHub Pages (deploy)

---

## Como Jogar

1. Escolha o nível de dificuldade (Fácil, Médio ou Difícil)
2. Clique em uma carta para virá-la
3. Clique em outra carta para tentar encontrar o par
4. Se as cartas forem iguais, elas permanecem viradas
5. Se forem diferentes, elas voltam ao estado original
6. Encontre todos os pares para vencer o jogo!

## Executar Localmente

Basta abrir o arquivo `index.html` no navegador:

```bash

cd jogo-memoria-gatinhos
open index.html
```

Não requer instalação de dependências — é 100% vanilla HTML/CSS/JS.
